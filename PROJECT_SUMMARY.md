## ðŸš€ Usage

### 1. Installation
```bash
pip install pydoc-markdown-nuxt
```

### 2. Configuration
Create `pydoc-markdown.yml`:
```yaml
renderers:
  - type: nuxt
    content_directory: content/docs
    default_frontmatter:
      layout: docs
      navigation: true
    pages:
      - title: API Documentation
        contents: ['*']
```

### 3. Generate Documentation
```bash
pydoc-markdown
```

### 4. Integrate with Nuxt.js
```javascript
// nuxt.config.js
export default {
  modules: ['@nuxt/content']
}
```
