# Installation

Install `pydoc-markdown-nuxt` using pip:

```bash
pip install pydoc-markdown-nuxt
```

## Requirements

- Python 3.7 or higher
- pydoc-markdown
- PyYAML
- Jinja2

The package will automatically install all required dependencies.
